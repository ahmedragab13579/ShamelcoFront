# UX Rules

## General

Users should never think about how to use the interface.

Everything should feel obvious.

---

Every page must contain

A clear title

A primary action

Visual hierarchy

Enough spacing

Responsive layout

---

Cards

Image

Title

Location

Rating

Price

CTA

---

Forms

Labels are mandatory.

Validation should appear immediately.

Error messages should explain the problem.

---

Buttons

Primary action

One per screen

Secondary actions

Outlined

Danger actions

Red

---

Search

Always visible

Fast

Easy to clear

Responsive

---

Loading

Skeleton screens

Not empty pages

---

Empty State

Illustration

Helpful message

Action button

---

Errors

Explain

Offer retry

Never expose technical details

---

Responsive

Desktop

Tablet

Mobile

Must provide the same experience.

Accessibility

Keyboard navigation

Visible focus states

Meaningful labels

Screen reader friendly
