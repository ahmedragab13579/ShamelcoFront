# Design Checklist

Before finishing any page, verify the following:

## Layout

- Hero section exists
- Visual hierarchy is clear
- Sections are separated
- No excessive empty space

## Components

- Consistent border radius
- Consistent shadows
- Proper spacing
- Responsive layout

## Typography

- Correct heading sizes
- Proper font weights
- Easy readability

## Colors

- Follow the 60-30-10 rule
- Primary CTA uses gold
- Contrast is sufficient

## UX

- Clear primary action
- Search is accessible
- Loading state exists
- Empty state exists
- Error state exists

## Motion

- Hover effects
- Smooth transitions
- No distracting animations

## Accessibility

- Keyboard accessible
- Focus states
- Proper labels
- Sufficient contrast

## Final Question

"Would I enjoy using this page if I had never seen it before?"
