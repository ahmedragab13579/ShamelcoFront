## Responsive Grid & Breakpoints

Mobile (< 640px)
1 Column Grid | Full width components

Tablet (768px - 1023px)
2 Column Grid | Standard spacing

Desktop (1024px+)
3 or 4 Column Grid | Max-width container (e.g., 1280px)

---

## Rules

Never start a page with cards.

Always create a visual entry point.

Group logically related sections together with the same background.

Use subtle background changes (shamelco-bg to surface) only to separate major context shifts, not strictly alternating every section.
